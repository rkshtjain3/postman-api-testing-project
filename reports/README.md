# Test Reports Directory

This directory contains generated HTML test reports from Newman executions.

## Report Files
- HTML reports are automatically generated when running tests
- Reports include detailed test results, request/response data, and performance metrics
- Each report is timestamped for easy identification

## Viewing Reports
Open any `.html` file in this directory with your web browser to view detailed test results.

## Note
Report files are excluded from git by default. Remove the reports/ entry from .gitignore if you want to commit reports to version control.
